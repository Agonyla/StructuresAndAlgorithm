### 链表类题目注意点

1. 如果比试中空间要求不严格，直接使用容器来解决链表问题
2. 如果比试中空间要求严格、或者在面试中面试官强调空间的优化，需要使用额外空间复杂度 `O(1)` 的方法
3. 最常用的技巧 - `快慢指针`
4. 链表类题目往往都是很简单的算法问题，核心考察点也并不是算法设计，是 `coding` 能力
5. 这一类问题处理多写多练没有别的应对方法

> 个人建议：
>
> 链表问题既然练的就是coding，那么不要采取空间上讨巧的方式来练习
